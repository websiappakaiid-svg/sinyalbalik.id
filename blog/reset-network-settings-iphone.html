Cara Reset Network Settings iPhone yang Benar

28 Mar 2026 4 menit baca

Kapan Harus Reset Network Settings?

Reset network settings adalah langkah troubleshooting yang sering disarankan kalau iPhone kamu mengalami masalah konektivitas. Tapi banyak orang nggak tau kapan harus pakai fitur ini dan apa efeknya.

Gunakan reset network settings kalau:

  • WiFi sering putus atau nggak mau konek
  • Sinyal seluler nggak stabil setelah update iOS
  • Bluetooth nggak bisa pairing
  • VPN bermasalah
  • Data seluler lambat padahal sinyal kuat

Apa yang Dihapus?

Reset network settings akan menghapus:

  • ✅ Semua password WiFi tersimpan
  • ✅ Pengaturan VPN
  • ✅ Pengaturan APN (access point)
  • ✅ Paired Bluetooth devices
  • ✅ Pengaturan proxy

Yang TIDAK dihapus:

  • ❌ Foto, video, app — aman
  • ❌ Kontak, chat WhatsApp — aman
  • ❌ File dan dokumen — aman
  • ❌ Pengaturan lain (display, sounds, dll) — aman
Pastikan kamu ingat password WiFi sebelum reset! Setelah reset, kamu perlu masukkan ulang password WiFi.

Step-by-Step Reset Network Settings

iOS 15 ke atas:

  1. Buka Settings
  2. Tap General
  3. Scroll ke bawah, tap Transfer or Reset iPhone
  4. Tap Reset
  5. Pilih Reset Network Settings
  6. Masukkan passcode
  7. Konfirmasi reset
  8. iPhone akan restart otomatis

iOS 14 ke bawah:

  1. Settings → General → Reset → Reset Network Settings

Setelah Reset, Lalu Apa?

  1. Reconnect ke WiFi (masukkan password lagi)
  2. Pair ulang Bluetooth devices
  3. Cek sinyal seluler — kalau sudah normal, masalah terselesaikan
  4. Kalau sinyal masih "No Service" → cek IMEI: Cara Cek IMEI

FAQ

Apakah reset network settings menghapus data?

Tidak. Yang dihapus hanya pengaturan WiFi, Bluetooth, VPN, dan APN. Foto, app, chat tidak terpengaruh.

Reset network bisa fix No Service?

Kalau penyebabnya network settings corrupt, bisa. Tapi kalau penyebabnya IMEI tidak terdaftar, reset network tidak akan membantu.

Berapa kali boleh reset network?

Tidak ada batasan. Tapi kalau sudah reset dan masalah tetap ada, berarti penyebabnya bukan di network settings.

Penutup

Reset network settings itu langkah troubleshooting yang ringan dan aman. Nggak menghapus data penting. Tapi ingat: ini bukan solusi untuk masalah IMEI. Kalau sinyal tetap mati setelah reset, cek IMEI kamu dulu.
